Accessibility statement for this guide

This page describes accessibility choices built into the site and the limits of local-storage convenience features.

How the site is designed

This site uses semantic HTML, keyboard-accessible navigation, visible focus states, readable contrast, reduced-motion support, and print-specific styles. Core reading and page navigation are available without JavaScript.

Notes and checklist convenience features

The notes page and checklist page use local storage so information can remain available when a person returns to the same browser on the same device. Those features are optional convenience tools.

Download support

The brochure is provided as a downloadable PDF asset. The notes page includes print-focused styling for user-entered notes.

Scope of this statement

This page describes the features implemented in this site. It does not make a blanket claim that every possible accessibility need is satisfied in all contexts.
